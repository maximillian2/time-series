Швидка допомога по програмі Time Series!

====== Кроки користування:
* Для користування програмою необхідно вибрати джерело даних. Це реалізовано завдяки введенню даних в самій програмі чи за допомогою файлу, в якому зберігаються дані.
* Після того, як джерело обрано, необхідно вибрати спосіб обчислення вхідного ряду.
* При потребі, є можливість включення / виключення сезонності.
* Натискання кнопки "Розрахувати" спричиняє появу вікна з графіком-результатом.

Для опису можливостей використовуйте пункт меню "Онлайн довідку".
